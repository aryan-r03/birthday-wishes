## ğŸš€ Quick Start

### Method 1: Direct File Opening (May Have Restrictions)

```bash
# 1. Download/clone project

cd only-for-you

# 2. Open in browser
# Double-click index.html
# OR
open index.html  # macOS

```

### Method 2: Using Python Server (Recommended)

```bash
# Navigate to project folder
cd only-for-you

# Start server (Python 3)
python -m http.server 8000

# Or Python 2
python -m SimpleHTTPServer 8000

# Open browser
# Visit: http://localhost:8000
```

### Method 3: Using Node.js

```bash
# Install http-server globally
npm install -g http-server

# Navigate to project
cd only-for-you

# Start server
http-server -p 8000

# Open browser
# Visit: http://localhost:8000
```

### Method 4: Using VS Code Live Server

1. Install "Live Server" extension in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"
4. Automatically opens in browser

## ğŸ¨ Customization Guide

### 1. Personalize Text Messages

**Location:** `index.html` (line ~95)

```javascript
const text = [
    "Your first line of text here",
    "Your second line here",
    "Your third line here",
    "Your fourth line here",
    "Your fifth line here",
    "Your sixth line here",
    "Your seventh line here",
    "Your eighth line here"
];
```

**Tips:**
- Each line fades in sequentially (3.5s per line)
- Keep lines under 60 characters for best display
